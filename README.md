# img-effects-noise-overlay

## Requirements

- [ImageMagick](https://imagemagick.org/)

## Usage

1. Place your input images (`.png`, `.jpg`, `.jpeg`) in the `__overlay__` directory.
2. Run `run.bat`.
3. Processed images will be saved in the `__overlay__fix__` directory.
